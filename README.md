## Hi there 👋
I love understanding how things work at the lowest level — from transistors to pixels on a screen.

🎨 Computer Graphics — fascinated by how pixels get to the screen, from rasterization pipelines to font rendering and clipping systems
⚙️ Low-level Programming — I want to know exactly what the machine is doing at every step, every byte intentional
🔌 Embedded — love the constraints, love the predictability, love being close to the hardware
